# Artifact Schemas

## tool-selection.json

Generated by `scripts/select_tools.py`. It records the user's per-run choice, active providers, unavailable selections, readiness reasons, and provider roles without storing credentials. Treat only entries with `active: true` as callable in this run.

## writing-selection.json

Generated by `scripts/select_writer.py`. It records the explicitly selected report-writing backend and model, supported roles, credential readiness, and the required service-region notice. `agent` means the current host Agent writes directly; any other choice means OpenRouter.

## research-map.md

```markdown
| ID | Subquestion | Type | Status | Claim IDs | Gap |
|---|---|---|---|---|---|
```

Use `required | optional | counterpoint` and `open | partial | covered | blocked`.

## query-log.md

```markdown
| Pack | Query ID | Query | Duty | Provider | Result count | Source IDs | Time | Notes |
|---|---|---|---|---|---:|---|---|---|
```

## evidence-ledger.md

```markdown
| Claim ID | Claim | Type | Supporting sources | Opposing sources | Confidence | Verification | Argument unit |
|---|---|---|---|---|---|---|---|
```

Claim type: `fact | source_view | analogy | inference | uncertainty`.

## outline.md

```markdown
# Argument Map
> Unordered coverage map, not a report outline.

| Unit ID | Required | Question/claim | Claim IDs | Relationships/tensions | Risk boundary |
|---|---|---|---|---|---|
```

